# 简介

虚数精灵是一款面向桌面自动化的可视化脚本编辑器。它通过节点编辑器的形式，让用户以拖拽连接的方式构建自动化流程，从而高效处理日常重复性任务。

无论是软件测试、数据录入，还是游戏辅助、界面操作自动化，虚数精灵都能帮助你快速搭建脚本，无需编写传统代码。

## 核心理念

虚数精灵的设计初衷是**降低自动化编程的门槛**。传统的自动化往往依赖编程语言或宏录制，灵活性不足且上手困难。虚数精灵采用可视化节点编辑模式，将功能封装为独立的"节点"，用户只需将节点拖拽到画布、连线定义数据流向和执行顺序，即可完成一个完整的自动化程序。

- **可视化编程**：无需手写代码，通过节点和连线直观表达逻辑
- **所见即所得**：脚本运行时可实时查看图像处理、绘制等操作的输出结果
- **灵活扩展**：内置丰富的节点库，覆盖图像采集、识别、键鼠控制、数据流转等常见场景

## 核心能力概览

| 能力 | 说明 |
| :--- | :--- |
| **多源图像采集** | 支持 USB 摄像头、采集卡、屏幕截图、窗口捕获及视频文件读取，灵活获取图像输入 |
| **图像处理** | 内置 OpenCV 图像处理能力，支持图像创建、编辑、绘制等操作 |
| **智能识别** | 集成模板匹配、文本识别（OCR）与 YOLO 对象检测，精准定位屏幕中的目标元素 |
| **键鼠录制与控制** | 支持键盘鼠标动作录制与回放，以及基于 KMBoxNet 协议（网络）和 Makcu 协议（串口）的远程键鼠控制 |
| **图形绘制** | 支持在图像上绘制文字、矩形框等图形元素，便于标注和输出 |
| **键盘鼠标控制** | 通过系统原生 API 精确模拟键盘输入和鼠标点击、移动、滚轮等操作 |
| **数据与日志** | 内置日志输出、JSON 数据处理与变量管理，方便调试和数据流转 |

## 适用场景

虚数精灵适用于广泛的自动化需求，包括但不限于：

- **桌面软件自动化**：自动填写表单、模拟用户操作进行软件测试
- **办公效率提升**：数据录入与格式转换
- **游戏辅助**：自动执行重复性游戏操作、图像识别触发动作
- **图像与视频批处理**：批量添加水印、格式转换、目标检测标注
- **远程设备控制**：通过网络或串口协议控制远程主机的键鼠操作
