# 更新日志

## v1.0.0-beta.6 - 2026-07-18

**✨ 新增特性 (Added)**

1. 运行时配置新增推理后端设备id选择功能。

**🛠️ 功能优化 (Changed)**

1. 对象检测节点性能提升。

**🐛 问题修复 (Fixed)**

1. 修复了对象检测只能使用固定640x640输入的YOLO模型问题。
2. 修复了使用KmBoxNet模拟键盘输入时，文本节点失效的问题。

## v1.0.0-beta.5 - 2026-07-04

**✨ 新增特性 (Added)**

**🛠️ 功能优化 (Changed)**

1. 现在允许粘贴“程序结束”节点至不包含该节点的画布。
2. 首次启动默认主题改为暗色。
3. 安装后首次启动将弹出用户协议对话框，点击同意后才能继续使用。
4. 更换了配置文件路径。
5. 数据存储格式调整：JSON对象的键全部改为camelCase式命名。
6. 对部分节点标题中显示的数据类型进行了多语言支持。

**🐛 问题修复 (Fixed)**

1. 修复编辑器内复制“程序”或“执行器”报错退出的问题。
2. 修复整数类型序列化不正确的问题。
3. 修复节点组拖拽角标显示异常的问题。
